# Mini Math AI Score — AI Math Grader

![React](https://img.shields.io/badge/React-61DAFB?style=flat-square&logo=react&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat-square&logo=javascript&logoColor=white) ![Gemini API](https://img.shields.io/badge/Gemini%20API-8E75B2?style=flat-square) ![Firebase](https://img.shields.io/badge/Firebase-FFCA28?style=flat-square&logo=firebase&logoColor=white) ![GSAP](https://img.shields.io/badge/GSAP-88CE02?style=flat-square) ![Framer Motion](https://img.shields.io/badge/Framer%20Motion-0055FF?style=flat-square) ![Recharts](https://img.shields.io/badge/Recharts-22B5BF?style=flat-square)

An AI-powered math problem grading tool with vector search. Built for a science fair project — includes full documentation (BAO_CAO_KHOA_HOC.md).

## What it does

- **AI Math Grading** — analyze and score solutions using Gemini API
- **KaTeX rendering** — proper math formula display
- **Vector search** — Qdrant integration for similar problem finding
- **Rich animations** — GSAP + Framer Motion for engaging UX
- **Excel export** — bulk score reports

## Getting started

```bash
git clone https://github.com/initforge/mini-math-ai-score.git
cd mini-math-ai-score
npm install
npm run dev  # Requires Gemini API key + Firebase config
```
