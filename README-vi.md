Công cụ chấm điểm bài Toán bằng AI với khả năng tìm kiếm vector. Dự án khoa học — có đầy đủ báo cáo (BAO_CAO_KHOA_HOC.md).

## Tính năng chính

- **Chấm điểm Toán AI** — phân tích và chấm lời giải bằng Gemini API
- **KaTeX rendering** — hiển thị công thức toán đúng chuẩn
- **Tìm kiếm vector** — tích hợp Qdrant tìm bài tương tự
- **Firebase Functions** — serverless backend xử lý AI
- **Animation phong phú** — GSAP + Framer Motion cho UX hấp dẫn
- **Xuất Excel** — báo cáo điểm hàng loạt
